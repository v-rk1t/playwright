# UI and API tests with Playwright 🎭

## Stacks 📚
- Node: https://nodejs.org
- TypeScript: https://www.typescriptlang.org
- Playwright: https://playwright.dev
- Zod: https://zod.dev
- Biome: https://biomejs.dev

## Commands ⌨️

### To run the tests
- npm run test
- npm run test:ui

### To view the test report
- npm run report
